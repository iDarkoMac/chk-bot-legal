# Privacy Policy for CHK Bot+

Last Updated: February 2026

CHK Bot+ ("the Bot") is a Discord bot designed to assist Kingshot players with reminders, events, gift codes, and related features.

## Information We Collect

CHK Bot+ may collect and store the following data:
- Discord User IDs
- Discord Server IDs
- Channel IDs
- Role IDs (for reminders and mentions)
- Gift code interaction logs

The Bot does NOT collect:
- Passwords
- Email addresses
- Private messages (unless required for a specific feature)
- Payment information

## How We Use Data

Data is used only to:
- Provide bot functionality
- Send scheduled reminders
- Manage events
- Improve performance and reliability

## Data Storage

Data is stored securely and is not sold, shared, or distributed to third parties.

## Data Removal

Server administrators may request data removal by removing the bot from the server.

## Contact

For questions regarding this Privacy Policy, contact the bot developer via Discord.
